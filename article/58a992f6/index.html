限制访问

python通过变量名的确定作用域

  • *xx: 公有变量
  • _x: 单前置下划线,私有化属性或方法，from somemodule import 禁止导入,类对象和子类可以访问
  • _xx：双前置下划线,避免与子类中的属性命名冲突，无法在外部直接访问(名字重整所以访问不到)
  • xx:双前后下划线,用户名字空间的魔法对象或属性。例如:init , __ 不要自己发明这样的名字
  • xx:单后置下划线,用于避免与Python关键词的冲突
    通过name mangling（名字重整(目的就是以防子类意外重写基类的方法或者属性)如：_Class__object）机制就可以访问private了

对象信息

函数type()可以判断是数据类型或者常数
函数instanceof()可以判断类型及其子类
函数dir()获得所有属性名和方法名组成d List

操作对象状态

对象中的属性和方法是动态的，可以添加，修改

setattr(对象实例,属性或方法名,属性的值或函数)

可以增加不存在的方法，将方法变为属性，替换已有方法等等
但也可通过实例.属性名=xx的方法绑定新变量或方法

getattr(对象实例,属性或方法名)

hasattr(对象实例,属性或方法名)

__slots__属性

可以限制实例能绑定的属性名，其值为属性名组成的元组
在子类中不起作用，若在子类中对其赋值，则为父类的__slots__与子类的__slots__之和

通过装饰器封装属性

@property

@属性名.setter

@属性名.deleter

具体用法

class Rect:
    def __init__(self,area):
        self.__area = area  #绑定属性
    @property
    def area(self):
        return self.__area
    @area.setter
    def area(self, value):
        self.__area = value
    @area.deleter
    def area(self):
        self.__area = 0
    
rect = Rect(30)
#直接通过方法名来访问 area 方法
print("矩形的面积是：",rect.area)   #30
rect.area = 90
print("修改后的面积：",rect.area)   #90
del rect.area
print("删除后的area值为：",rect.area)   #0

定制类

开发者看到的字符串

repr()

iter

返回迭代器用于遍历某个container，对于可迭代对象，返回self

next

对可迭代对象使用for…in… ，python会先调用iter()得到迭代器，再不断调用迭代器的next()得到下一个值，直到遇到StopIteration错误。故__next__应当返回一个container中的对象，并在遍历完成时raise StopIteration

定制列表行为

getitem(self, key)

setitem(self, key, value)

delitem(self, key)

getattr(self, name)

在访问对象不存在的属性时返回某个值或函数
name是发生AttributeError时调用的属性名
根据name做出不同的反应，只要写了这个方法，遇到属性不存在，默认返回None
可以利用这个动态给对象增加属性

call

将实例当作函数调用时的行为
使用callable(实例)可判断实例是否已实现__call__方法
